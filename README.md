# NodeJs Express Typescript Boilerplate 

---

## To setting up the project

Clone the repo

```bash
https://github.com/mikaeelkhalid/express-typescript-boilerplate.git
```

cd into the cloned repo and run:

```bash
npm i
```

Start Project in Dev

```bash
npm run dev
```

Build the project

```bash
npx tsc
```

Run Build project

```bash
npm start
```
